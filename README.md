# Maggu Brothers

Static premium website for Maggu Brothers — bespoke automotive customisation, Chandigarh.

## Pages

- `index.html` — Homepage with responsive video hero
- `services.html` — Three discipline showcase (Audio, Interiors, Wraps)
- `gallery.html` — Filterable archive of selected builds
- `booking.html` — Studio reservation flow with live calendar

## Stack

Pure static — no build step.

- HTML5
- Tailwind CSS (CDN)
- Fraunces + Geist + Geist Mono (Google Fonts)
- Vanilla JS

## Shared assets

- `site.css` — design tokens, layout, components, mobile breakpoints
- `site.js` — preloader, custom cursor, scroll progress, reveal animations, mobile hamburger menu, responsive video loader

## Local preview

Open `index.html` in any modern browser.

## Deploy

Drop the entire repo on any static host (Netlify, Vercel, Cloudflare Pages, GitHub Pages). No build required.

## Before going live

Replace `https://wa.me/` with your actual WhatsApp number in:
- `index.html`
- `services.html`
- `gallery.html`
- `booking.html`
